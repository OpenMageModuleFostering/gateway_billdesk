BillDesk Payment Gateway extension is a module for the Magento e-commerce platform based websites. 
Developed by Mahiti Infotech, Bangalore. BillDesk, being one of the most popular payment gateways in India, 
has been providing secure, fast and reliable web and mobile payment solutions to Businesses, NGOs and the Government. 
This module is aimed at providing easy integration of Billdesk Payment Gateway with Magento based shopping carts.
